## 학습 목표

이 레슨을 마치면 다음을 할 수 있게 됩니다.

- Claude의 추가 제품 — **Claude Code, Claude for Slack, Claude for Excel, Claude for Chrome** — 을 언제 써야 할지 이해한다

## 도입

코스 도입부에서 말했듯, **Claude는 인텔리전스(intelligence)**입니다. [Claude.ai](https://claude.ai/)는 그것과 일하는 *하나의 방법*일 뿐입니다.

Claude는 이미 일하고 있는 환경 안으로 직접 들어갈 수 있도록, 여러 특화 도구로도 제공됩니다. 이 레슨에서는 워크플로우와 use case에 맞춰 설계된 **추가 4가지 도구**를 소개합니다.

## Claude Code

<!-- youtube: s-avRazvmLg -->

Claude Code는 여러분이 일하는 곳 — **터미널·IDE·브라우저·심지어 Slack** — 어디서든 동작하는 **에이전트형(agentic) 코딩 도구**입니다. 코드베이스를 이해하고, 명령을 실행하며, 자연어로 전체 개발 워크플로우를 처리합니다.

**Claude Code를 쓰는 게 좋을 때:**

- 필요한 기능을 평이한 영어로 설명해 Claude가 코드를 작성하고, 테스트를 돌리고, 커밋까지 만들게 하고 싶을 때
- 에러 메시지를 붙여 넣고 Claude가 코드베이스를 분석해 원인을 찾아 고치도록 하고 싶을 때
- 낯선 코드베이스에서 각 부분이 어떻게 맞물리는지 자연어로 묻고 싶을 때
- lint 오류 수정, 머지 컨플릭트 해결, 릴리스 노트 작성 같은 반복 작업을 자동화하고 싶을 때
- 별도 인터페이스로 전환하지 않고, 기존 IDE·개발 도구와 함께 **터미널에서** 작업하는 게 편할 때

## Claude in Slack

<!-- youtube: tI1uzSJuSxc -->

Claude는 Slack에 직접 통합됩니다. 채널·스레드 안에서 도움을 받거나, Slack의 맥락을 Claude 대화로 끌어올 수 있습니다.

**Claude in Slack을 쓰는 게 좋을 때:**

- Slack을 떠나지 않고 메시지에 답장 초안을 짜거나, 긴 스레드를 요약하거나, 복잡한 논의를 분해하고 싶을 때
- 미팅 준비를 위해 워크스페이스 전반의 관련 대화·공유 문서를 모아 보고 싶을 때
- 새 팀에 합류해 진행 중인 프로젝트를 채널 히스토리로 빠르게 파악하고 싶을 때
- 버그 리포트나 기능 논의에서 바로 코딩 작업을 넘기고 싶을 때 — **`@Claude`** 만 태그하면 주변 맥락을 가지고 Claude Code 세션을 띄워 줍니다
- 대화 도중 산업 트렌드, 기술 개념, 회사 정보에 대한 답을 빠르게 얻고 싶을 때

## Claude for Excel

<!-- youtube: 8ZRTSIRWLu4 -->

Claude for Excel은 **Microsoft Excel 사이드바** 형태로 Claude를 직접 불러와, 대화로 스프레드시트를 분석·이해·수정할 수 있게 해 줍니다.

**Claude for Excel을 쓰는 게 좋을 때:**

- 여러 탭으로 구성된 복잡한 워크북에서 시트를 가로지르는 수식·계산 흐름이 어떻게 동작하는지 이해하고 싶을 때
- 모델의 가정·입력값을 업데이트하면서도 수식 의존 관계를 유지해야 할 때
- `#REF!`, `#VALUE!`, 순환 참조(circular references) 같은 오류를 추적해 원인부터 고치고 싶을 때
- 새 스프레드시트를 만들거나 기존 템플릿에 데이터를 채우면서도 적절한 수식 구조를 유지하고 싶을 때
- 피벗 테이블이나 차트로 데이터를 빠르게 시각화하고 싶을 때

## Claude for Chrome

<!-- youtube: IypXvHej9eY -->

Claude for Chrome은 **Google Chrome의 사이드바**로 Claude를 추가해 주는 **브라우저 확장**입니다. 여러분이 작업 중인 화면을 관찰하고 브라우저 안에서 직접 행동을 수행할 수 있습니다.

**Claude for Chrome을 쓰는 게 좋을 때:**

- 브라우징 중에 기사·논문·웹 페이지를 요약하고 싶을 때
- 이메일 답변 초안 작성·받은 편지함 정리에 도움이 필요할 때
- 반복적인 폼 입력을 자동화하고 싶을 때
- 단계별로 직접 클릭할 필요 없이 웹사이트 기능을 테스트하거나 다단계 워크플로우를 따라가고 싶을 때
- 탭을 옮겨 다닐 때도 맥락을 유지해 주는 브라우징 어시스턴트가 필요할 때 — 사내 도구·CRM·대시보드처럼 특수한 환경에서 맥락을 끌어오기 좋습니다

> **중요.** Claude for Chrome은 현재 **research preview** 단계입니다. Anthropic은 신뢰할 수 있는 사이트에서 **위험도가 낮은 작업**부터 사용할 것을 권장합니다. 결제·개인정보 공유처럼 위험한 행동 전에는 권한을 묻고, **금융 서비스·성인 콘텐츠** 같은 카테고리는 기본 차단됩니다.

## 정리

각 도구는 Claude의 능력을 여러분이 실제로 일하는 환경으로 확장합니다.

| 도구 | 적합한 작업 | 동작 환경 |
|---|---|---|
| **Claude.ai** | 일반 작업, 리서치, 작성, 분석, 파일 생성 | 웹·데스크톱·모바일 앱 |
| **Claude Code** | 소프트웨어 개발, 코드베이스 탐색, git 워크플로우 | 터미널/CLI, IDE, 브라우저 |
| **Claude Cowork** | 복잡한 다단계 작업: 리서치 브리프, 문서 작성, 파일 정리, 데이터 분석 | 데스크톱 (모바일은 Dispatch 경유) |
| **Claude / Claude Code in Slack** | 팀 협업, 미팅 준비, 맥락 안에서 빠른 답변 | Slack 워크스페이스 |
| **Claude for Excel** | 스프레드시트 분석, 재무 모델링, 수식 디버깅 | Microsoft Excel 사이드바 |
| **Claude for Powerpoint** | 슬라이드 작성, 프레젠테이션 편집, 포맷·디자인 | Microsoft PowerPoint 사이드바 |
| **Claude for Chrome** | 웹 리서치, 이메일 관리, 브라우저 자동화 | Chrome 브라우저 사이드바 |
